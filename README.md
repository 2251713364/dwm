# dwm
## patches

### autostart
开启dwm时自动运行一些脚本
### dwm-alpha
状态栏透明
### awesomebar
让状态栏显示窗口名字
### fullscreen
让窗口全屏
### hide_vacant_tags
只显示有窗口的标签
### pretag
不同的标签可以使用不同的排列方式
### noborder
状态栏视觉效果
### viewontag
将窗口移到另一个标签时，切换到相应的标签
### rotatestack
转换某一标签内窗口的顺序
### scratchpad
临时打开一个小窗口
### vanitygaps
给窗口之间增加一些小空隙
